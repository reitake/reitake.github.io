# Batlefiled 5  

## 枪

- 机枪伤害衰减：9m 25 → 50m 20 → 70m 18.5

### 突击兵  

- 格韦尔1-5突击步枪
    + 适合新手，可单发/全自动
    + 伤害不如stg44，射速比stg44快，近距离对战优势大
    + 配合三倍镜在中近距离有稳定输出

### 医疗兵  

- 索米KP/-31冲锋枪
    + 腰射可控性强，可单发/全自动，770射速
    + 虽然伤害降低，但子弹初速度高，中近距离输出整体加强
    + 建议新手走大弹夹，熟悉手感再换腰射六
- 斯登冲锋枪
    + 射速540，全自动
    + 属性平庸，适合新手适应冲锋枪
- MP40冲锋枪
    + 射速540
    + 射速一般，但腰射准度和开镜准度借专长提高很多，适合大图中距离交火

### 支援兵  

- 刘易斯机枪
    + 精度和爆发不出色，但满级97子弹弹夹，可提供长时间的火力支援
    + 后坐力略大于KE7，射速略高于布伦，但大幅低于升级后的KE7
    + 长时间射击没有枪管冷却
    + 三脚架搭配三倍镜是和中远距离
- KE7
    + 上手容易
    + 弹夹稍小，升级后属性出色
- MG34
    + 固定式机枪，缺乏机动性，遭遇战反应力差
    + 初始50发弹夹，670射速。升级后最大100的弹链，770的射速，不如MG42
- BERN布伦
    + 平庸，射速慢，弹夹小
    + 精度高，后坐力低
    + 推荐打法：升级后坐力，带三倍镜
- MG42
    + 大概是最好用的
    + 初始981射速，可升级成1200射速或250弹链
- 12G霰弹枪
    + 射速慢，弹一枪致死
    + 后坐力动画幅度太大，遮挡屏幕
    + 伤害8m 5→9m 3→24m 2
- M30
    + 强势，伤害 12m 5→ 13m 3→24m 2
    + 自带步枪弹，配2倍镜机瞄，可以当半个狙

### 侦察兵  

- 格位尔M95/40步枪
    + 伤害80→60
    + 在中近距离的范围内有较小的子弹掉落
    + 伤害比老李高，适合换手枪补刀
    + 综合性能较高，适合玩家过度使用
    + 近距离能打出80伤害，能打出击杀助攻，还这个距离很危险
- 老李
    + 初始武器，但不一定适合新手
    + 伤害60→55
    + 射速快，但子弹初速慢，下坠大
    + 技术上限高，遇到较多敌人时候，枪法硬还是有一战之力
- KragJorgensen
    + 伤害54→54
    + 有个低阻弹专长，减少下坠，提高远距离爆头率
    + 缺点伤害低，第二枪打到手腿可能50伤害打不死
- Kar98K
    + 伤害75→60
    + 弹速快，还有低阻弹专长，下坠非常小
    + 缺点射速慢

## 武器专长  

- 快速瞄准：瞄具瞄准的速度加快33%
    + 适合搭配狙击枪，提高射速，因为需要切镜上弹
- 可变瞄准：远距离狙击时可以变更武器瞄准
    + 没什么用，不过有时候不得不点来激活下一个专长
- 轻量化枪托：武器瞄准时的移动速度加快60%
    + 高倍镜（6）时合适，低倍镜时可能不便于瞄准
